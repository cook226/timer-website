# Timer App

A simple, clean interval timer built with modern web technologies. Perfect for workout intervals, cooking timers, or any timed repetitive tasks.

## 🔗 [Live Demo](https://cook226.github.io/timer-website/)

## ✨ Features

- Set custom timer duration in seconds
- Configure number of loops (or run infinitely)
- Pause and resume functionality
- Visual and audio feedback when timer completes
- Clean, responsive design
- Loop counter display
- Mobile-friendly interface

## 🛠️ Technologies Used

- HTML5
- JavaScript (ES6+)
- TailwindCSS for styling
- SVG icons for UI elements
- Browser Audio API

## 🔧 Technical Features

- DOM manipulation
- Event handling
- Interval timing
- Audio playback control
- State management
- Responsive design
- CSS Flexbox
- SVG integration

## 📱 Usage

1. Enter desired time in seconds
2. (Optional) Enter number of loops
3. Click Play to start timer
4. Use Pause to temporarily stop
5. Use Stop to reset completely
6. Audio beep signals end of each interval
